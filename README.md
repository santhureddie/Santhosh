# F1 Three.js Portfolio Starter

An interactive Formula-1 themed portfolio inspired by game-like scrolling portfolios.

## Run locally

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## Personalize content

Edit `portfolio-data.js` and replace:

- Work experience bullets
- Skills stack
- Project links
- Contact details

## Controls

- `← / →` or `A / D`: steer around the track
- `↑ / ↓` or `W / S`: boost / brake
